# Как пользоваться 
Расширение добавляет в предпросмотр маркдауна несколько удобных штук: стили под GitHub, свои блоки для предупреждений и спойлеров, анимацию и эмодзи.

# Установка
Если у вас есть .vsix файл:

Открываете VS Code, жмёте Ctrl+Shift+X (расширения)

Три точки в правом верхнем углу → Install from VSIX...

Выбираете файл.

# Если ставите из исходников:

npm install

npm run compile

Нажимаете F5 (откроется новое окно с вашим расширением).
